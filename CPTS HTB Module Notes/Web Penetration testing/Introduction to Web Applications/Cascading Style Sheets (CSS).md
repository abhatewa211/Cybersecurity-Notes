- **CSS (Cascading Style Sheets)** is a **stylesheet language** used with HTML.
    
- It is used to **format and style HTML elements**.
    

✔ Defines:

- Colors
    
- Fonts
    
- Layout
    
- Spacing
    
- Animations
    

✔ Multiple versions exist → each adds new features

---

## 🧱 Basic CSS Example

```css
body {
  background-color: black;
}

h1 {
  color: white;
  text-align: center;
}

p {
  font-family: helvetica;
  font-size: 10px;
}
```

### 📌 Explanation:

- `body` → sets background color
    
- `h1` → styles headings
    
- `p` → styles paragraphs
    

✔ Applies to **all matching HTML elements**

## 🔤 CSS Syntax

```css
element {
  property: value;
}
```

### 📌 Structure:

- **Selector** → HTML element (e.g., `p`, `h1`)
    
- **Property** → What to change
    
- **Value** → How to change it
    

✔ Written inside `{ }`  
✔ Each rule ends with `;`
