- Mainly used for:
    
    - **Web development**
        
    - **Mobile development**
        

✔ Runs mostly on the **front end (browser)**  
✔ Can also run on **back end** using **Node.js**

## 🎯 Role of JavaScript

- HTML → Structure
    
- CSS → Styling
    
- **JavaScript → Functionality**
    

✔ Without JavaScript:

- Web pages are mostly **static**
    
- No interactivity

# 🧱 JavaScript in HTML

---

## 🔹 Using `<script>` Tag

```html
<script type="text/javascript">
..JavaScript code..
</script>
```

---

## 🔹 Linking External JS File

```html
<script src="./script.js"></script>
```

✔ Keeps code **modular and clean**

---

## 🔹 Basic Example

```javascript
document.getElementById("button1").innerHTML = "Changed Text!";
```

### 📌 What it does:

- Finds element with **id = button1**
    
- Changes its text to **"Changed Text!"**
